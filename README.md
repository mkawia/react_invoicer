React invoicer
========================

A demo invoicer making app made in reactjs with a nodejs backend

Libraries used
------------

  * Reactjs;
  * [Koajs][1];
  * [ant design component library][2].
  * [pouchdb][3]



Requirements 
------------
  * Nodejs 10 orhigher;
  * A computer

Usage
------------

Run the production script on the root folder
```bash
$ npm run prod
```

------------

Run npm install on the root folder
```bash
$ npm install
```

```bash
$ npm install
```
